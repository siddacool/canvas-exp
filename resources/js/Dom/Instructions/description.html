About

Tower of babel is a mock 3d illustration showing the capabilities of WebGL. Note: it is not a source of information regarding Tower of Babylon.

Usage and Specs

Instructions on how to use Tower of Babel and the reuired specs to use it.

Controls
  • Hover over Mesh - to see Informative Tooltip
  • Click + Drag - to Rotate and move around
  • Wheel - to Zoom in/out
  • Double click - to see Extended Information
Specs Requirements
  • Hardware + Browser supporting WebGL
  • Computer Mouse/Touchpad/Trackpad
